# Deprecated

These install scripts are **no longer the recommended way** to set up a
machine. Package installation, SSH key setup, and (eventually) dotfile
deployment are handled by a separate **Ansible** project.

The scripts here are retained as historical reference and as a manual
fallback for environments where Ansible is not yet available. They are
not kept up to date.

If you are setting up a new machine, use the Ansible project instead.
